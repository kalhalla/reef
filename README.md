# REEF

A voxel coral reef toy in Three.js. Phase 1: seabed, orbit camera, placeholder coral growth.

## Run

    npm install
    npm run dev

## Build

    npm run build

Output is in `dist/`. Deploy to Vercel as a static site.

## Controls

- Mouse drag / one-finger drag: orbit
- Scroll / pinch: zoom
- Right-click drag: pan
- Click / tap on the seabed: plant coral
